What is IDB?

IDB (identibooru) is a custom Danbooru-style imageboard for user profiles rather than images.
It allows users to describe themselves in detail and to easily find eachother per interest/type.

Users can tag themselves with (currently) up to 2000 custom tags and publicize an accurate or approximate
location of themselves on the searchable tag map.

All users are tagged by one of the three safety ratings (rating:safe, rating:questionable, or rating:explicit),
which describe the tags and type of content their profile will likely contain.

Read more: User Accounts

Currently, guests can only search for rating:safe profiles.
